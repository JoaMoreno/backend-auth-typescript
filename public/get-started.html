Routes

Todas las rutas empiezan por: https://apirest.glitch.me/api

Metodos

GET

/products

https://apirest.glitch.me/api/products

Retorna un objeto JSON con todos los productos en la DB

POST

/products

https://apirest.glitch.me/api/products

Guarda un nuevo producto, requiere que se le envie un JSON

{

"name": "Product Name"

"description": "Product Description"

"precio": "Product Price"

"imgURL": "Product Image"

}

POST

/products/id

https://apirest.glitch.me/api/products/ + id

Retorna un unico producto, se require el id del producto en la URL

PUT

/products/id

https://apirest.glitch.me/api/products/ + id

Edita un producto, se require el id del producto en la URL y el objeto JSON con los datos

DELETE

/products/id

https://apirest.glitch.me/api/products/ + id

Elimina un producto, se require el id del producto en la URL

Metodo POST para eliminar DEPRECADO
